# Projeto: Hot Family

> Desenvolvido por Brazu Marketing Digital – https://agenciabrazu.com.br

---

## 📝 Descrição
Este projeto foi desenvolvido para Hot Family, com foco em desempenho, responsividade e integração com ferramentas de métricas e rastreamento.

---

## 🚀 Tecnologias usadas
- React / NextJS
- JavaScript
- Vercel (deploy)

---

## 🧠 Funcionalidades
- Eventos personalizados (clique WhatsApp, envio de formulário, etc.)
- Design responsivo e limpo

---

## 🔧 Como rodar localmente

bash
# Clone o repositório
git clone https://github.com/danielcerqueira7747/hotfamily.git

# Acesse a pasta
cd hotfamily

# Instale as dependências
npm install

# Rode localmente
npm run dev
